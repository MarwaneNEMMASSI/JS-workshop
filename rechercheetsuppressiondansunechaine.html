Recherche et suppression dans une chaine

Enoncé

Ecrire un programme Javascript qui demande a l'utilisateur d'écrire une phrase dans une zone de texte et un mot dans une autre zone de texte. Le clique sur le bouton recherche affiche un message indiquant si le mot éxiste dans la phrase ou non. Le clique sur le boutton supprimer permet d'effacer toutes les occurences du mot dans la phrase (si le mot éxiste biensur)
